# Empty Miles

> ⚠️ **Early-stage prototype — work in progress.** This project is under active 
> development and not yet production-ready. Expect breaking changes.

## What is this?
A fleet management tool designed to help independent truck drivers reduce deadhead 
(empty) miles by optimizing load planning and route decisions.

## Current Status
- Single-file HTML prototype (no framework, no backend)
- Core 9-tab UI in place
- Backend, authentication, and data persistence coming in a future sprint

## Roadmap
